üöÄ How to Use Multi-Floor Design

  1. Start with the default Ground Floor (Floor 1)
  2. Add rooms to the current floor using room templates
  3. Click the "+" button in floor tabs to add a new floor
  4. Switch between floors using the numbered tabs
  5. Rename floors by editing the floor name input
  6. Move rooms between floors using the room properties panel
  7. Copy selected rooms to other floors using copy buttons
  8. Remove floors (except the last one) using the Remove Floor button
  9. View floor-specific statistics in the plan stats section
  10. Save and download plans with complete floor information
