üéØ How It Works

Stage Filtering Process:

  1. Project Selection: When contractor selects a project, system fetches existing payment requests
  2. Paid Stage Detection: API returns list of stages with status = 'paid'
  3. Grid Filtering: Only stages NOT in the paid list are displayed
  4. Visual Feedback: Shows info about paid stages and remaining available stages
  5. Completion Check: If all stages are paid, shows completion message
