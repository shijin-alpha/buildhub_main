üéØ How It Works

Contractor Workflow:

  1. Navigate to Custom Payment: Go to Progress Updates ‚Üí Custom Payment
  2. Select Project: Choose the project needing additional payment
  3. View Budget Summary: See current budget status and overrun
  4. Fill Request Details: Title, reason, amount, description
  5. Set Priority: Choose urgency level and category
  6. Submit Request: Send to homeowner for approval

Budget Tracking:

  1. Original Estimate: Base cost from accepted estimate
  2. Stage Payments: Regular construction stage payments
  3. Custom Payments: Additional payments for extra work
  4. Total Cost: Sum of all payments requested/paid
  5. Budget Analysis: Overrun/underrun calculation and percentage
